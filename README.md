# 📊 HR Analytics Lab 2 using Python

## 📌 Project Overview
**HR Analytics Lab 2** is a Python-based project designed to analyze employee-related data and generate meaningful insights. It demonstrates how data analysis techniques can be applied to HR datasets for better decision-making and understanding workforce trends.  

This project covers:
- Employee data analysis  
- Data cleaning and preprocessing  
- Visualization of HR metrics  
- Insight generation for HR decisions  

---

## 🚀 Features
- **Data Analysis** → Extracts useful insights from employee datasets and identifies workforce trends  
- **Data Cleaning** → Handles missing values and inconsistencies, preparing structured datasets for analysis  
- **Data Visualization** → Uses charts and graphs to represent HR metrics, making insights easy to interpret  
- **Insight Generation** → Supports data-driven HR decisions and provides summary findings for workforce management  

---

## 🛠️ Tech Stack
- **Python**  
- **Pandas** – Data manipulation & analysis  
- **NumPy** – Numerical computations  
- **Matplotlib / Seaborn** – Data visualization
